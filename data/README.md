This folder contains the dataset and processed splits used in the project.

## Raw dataset (not tracked in Git — download separately)

**File**: `telco_churn.csv`  
**Source**: [Kaggle — Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)  
**Original filename**: `WA_Fn-UseC_-Telco-Customer-Churn.csv`

**To set up:**
1. Download the dataset from Kaggle (free account required)
2. Rename it to `telco_churn.csv`
3. Place it in this `data/` folder
4. Run notebooks in order starting from `01_eda.ipynb`

## Processed files (generated by notebooks)

These are created automatically when you run the notebooks:

| File | Generated by | Description |
|---|---|---|
| `X_train.csv` | `02_modelling.ipynb` | Training features (80% split) |
| `X_test.csv` | `02_modelling.ipynb` | Test features (20% split) |
| `y_train.csv` | `02_modelling.ipynb` | Training labels |
| `y_test.csv` | `02_modelling.ipynb` | Test labels |
| `shap_importance.csv` | `05_shap_explainability.ipynb` | Feature importance from SHAP |

## Dataset description

7,043 telecom customers with 21 features:

- **Demographics**: gender, senior citizen, partner, dependents
- **Account**: tenure, contract type, paperless billing, payment method, monthly charges, total charges
- **Services**: phone, multiple lines, internet, online security, online backup, device protection, tech support, streaming TV, streaming movies
- **Target**: Churn (Yes/No)

**Class distribution**: 73.5% No Churn / 26.5% Churned (imbalanced)
